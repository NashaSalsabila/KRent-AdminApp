# KRent-AdminApp

<h2> Penjelasan Program </h2>

Program ini membantu Admin sebuah tempat penyewaan sepeda motor untuk 
memonitor pelanggan nya. Tersedia fitur untuk save, delete, clear, dan print untuk mencetak hasilnya.
